## 🇺🇸 English

A fully hand-coded landing page built as part of [The Odin Project](https://www.theodinproject.com/) Foundations curriculum. The goal was to recreate a provided design from scratch using only HTML and CSS, with a focus on Flexbox layout.

### 🧠 Concepts Practiced

- Flexbox layout (containers, alignment, spacing)
- Page sectioning with semantic HTML
- CSS custom properties and color theming
- Typography with Google Fonts
- Linking external stylesheets
- Recreating a design from a visual reference

### 📁 Project Structure

```
odin-landing-page/
├── index.html
└── style.css
```

### 🚀 How to View

**Live:** [rkeppler42.github.io/odin-landing-page](https://rkeppler42.github.io/odin-landing-page)

**Locally:**

```bash
git clone https://github.com/rkeppler42/odin-landing-page.git
cd odin-landing-page
# Open index.html in your browser
```
